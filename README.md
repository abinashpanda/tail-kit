UI kit built using `tailwindcss`

## Install

We use `yarn` to install the dependencies.

```sh
yarn install
```

## Scripts

- `yarn storybook` - Run storybook
- `yarn test` - Run test in watch mode
- `yarn test:ci` - Run test with ci mode and run coverage
